
# Azure Networking Labs

This folder contains my hands-on Azure networking labs created while preparing for the Microsoft AZ-104 certification.

## Goals

- Learn Azure networking concepts
- Build practical Azure networking skills
- Understand Azure networking best practices
- Improve troubleshooting skills
- Gain real-world Azure experience

## Topics

- Virtual Networks (VNets)
- Subnets
- Network Security Groups (NSGs)
- Virtual Network Peering
- Azure Load Balancer
- NAT Gateway
- SNAT Exhaustion
- Private Endpoints
- Private DNS
- User Defined Routes (UDRs)
